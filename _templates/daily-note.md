# Daily Note — alias

Alias de compatibilidad para referencias históricas a `daily-note`. La plantilla canónica actual para nota diaria es [[nota-diaria]].

## Uso

No crear notas nuevas desde este archivo salvo que una herramienta externa espere explícitamente `daily-note.md`. Para uso normal, copiar [[nota-diaria]].
