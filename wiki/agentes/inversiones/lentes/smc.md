# SMC Lens

Evalúa conceptos SMC con cautela: liquidez, barridas, imbalance, order blocks. No debe vender humo.

## Checklist

- ¿Hay sweep claro de liquidez?
- ¿Hay displacement real o solo interpretación forzada?
- ¿El order block está alineado con estructura mayor?
- ¿SMC contradice niveles básicos?

## Output

SMC apoya / no aporta / contradice + nivel clave.

## Reglas

- Este lente no decide.
- Debe separar hechos de supuestos.
- Debe indicar si no hay datos suficientes.
- Debe señalar contradicciones.
