# Lentes / checklists — Sistema de inversiones v2

Estos archivos son **lentes/checklists**, no agentes. Informan a los Decision Leads y a Risk Manager, pero no deciden.

## Regla

Un lente debe devolver evidencia breve, señal, nivel de confianza y contradicciones. Si empieza a emitir recomendaciones finales, está violando su rol.
