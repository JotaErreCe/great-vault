# Technical Analysis Lens

Evalúa señales técnicas para swing: tendencia, medias, RSI/MACD, niveles, ruptura y momentum.

## Checklist

- ¿Tendencia clara?
- ¿Precio sobre/bajo medias relevantes?
- ¿RSI/MACD apoyan o advierten sobreextensión?
- ¿Nivel de entrada/invalidation objetivo?

## Output

Setup técnico válido / débil / inválido.

## Reglas

- Este lente no decide.
- Debe separar hechos de supuestos.
- Debe indicar si no hay datos suficientes.
- Debe señalar contradicciones.
