# Primer reporte simulado — Jordan

> Simulación de formato. No usa todavía portafolio real actualizado ni precios live. Sirve para validar estructura, tono y flujo de decisión.

## 1. Resumen ejecutivo

JR tiene dos capitales separados para Fase 1:

- USD 100 para trading de bolsa.
- USD 100 para trading de cripto.

Con perfil de riesgo 4.75/10, el objetivo del primer mes no es “pegar un home run”; es aprender proceso, registrar tesis y evitar errores tontos.

Conclusión simulada: **no hay obligación de entrar de inmediato**. Esperar, dividir entrada o tomar posición simbólica son decisiones válidas.

## 2. Capital de trading de bolsa — USD 100

### Enfoque sugerido

Para el primer mes, Jordan analizaría tres caminos:

1. ETF amplio o sectorial líquido.
2. Acción individual de calidad con catalizador claro.
3. Mantener efectivo si el mercado está extendido o la tesis no es clara.

### Criterios de entrada

- Tesis explicable en 2–3 frases.
- Riesgo máximo claro antes de entrar.
- Nivel de invalidación definido.
- No entrar solo por FOMO o redes sociales.

### Ejemplo de estructura de trade

| Campo | Ejemplo simulado |
|---|---|
| Activo | ETF o acción líquida |
| Monto máximo | USD 25–50 inicialmente |
| Entrada | gradual, no todo de golpe si hay volatilidad |
| Invalidación | pérdida de soporte / tesis rota / noticia adversa |
| Salida esperada | objetivo parcial o revisión semanal |
| Aprendizaje | registrar si la entrada fue razonada o emocional |

### Traducción a lenguaje simple para JR

No queremos apostar los USD 100 como si fueran casino. Queremos usar ese dinero como “laboratorio barato”: entrar solo si la idea se entiende, si sabemos cuánto podemos perder y si hay una razón mejor que “vi que subió”.

## 3. Capital de trading de cripto — USD 100

### Enfoque sugerido

Satoshi/Jordan compararían:

1. BTC/ETH como base cripto menos especulativa.
2. Altcoin con narrativa fuerte, pero solo si tokenomics/supply no contradicen la subida.
3. Efectivo/stablecoin si el mercado está eufórico o sin entrada clara.

### Criterios especiales cripto

- Revisar supply/unlocks.
- Revisar volumen y liquidez.
- Revisar si hay entrada de tokens a exchanges.
- Separar narrativa real de hype.
- Evitar altcoins sin liquidez o movidas solo por influencers.

### Ejemplo de estructura de trade

| Campo | Ejemplo simulado |
|---|---|
| Activo | BTC/ETH o altcoin justificada |
| Monto máximo | USD 20–40 si riesgo medio; menos si altcoin |
| Entrada | gradual o esperar pullback |
| Invalidación | pérdida de nivel clave / unlock adverso / tesis rota |
| Salida esperada | toma parcial o revisión semanal |
| Aprendizaje | registrar si el riesgo de supply estaba entendido |

### Traducción a lenguaje simple para JR

En cripto el precio puede subir muy rápido, pero también puede caer igual de rápido. Antes de entrar, Satoshi debe explicar si la moneda realmente tiene demanda o si solo está subiendo por emoción. Si no se entiende, no se compra.

## 4. Qué información necesita Jordan de JR

Para activar análisis real, Jordan necesita:

### IBKR

- posiciones actuales;
- cantidad;
- precio promedio/cost basis;
- valor actual;
- P/L si aparece;
- cash disponible si aplica;
- fecha de corte.

### Binance

- activos actuales;
- cantidad;
- valor actual en USD;
- precio promedio si existe;
- staking/earn si aplica;
- stablecoins/cash disponible;
- fecha de corte.

### Preferencias

- si los USD 100 bolsa y USD 100 cripto se aportarán mensualmente o solo este primer mes;
- si quiere permitir posiciones simbólicas de aprendizaje de USD 5–10;
- si prefiere que Jordan sea conservador al inicio y espere más.

## 5. Decisión práctica simulada

Para el primer mes, sin datos reales actualizados, Jordan no recomendaría entrar aún.

Acción sugerida:

1. Cargar portfolio real.
2. Crear Google Sheet o equivalente CSV temporal.
3. Definir una primera idea de bolsa y una primera idea cripto.
4. Exigir tesis + invalidación antes de usar capital.

Frase Jordan: **Esperar también es una decisión.**
